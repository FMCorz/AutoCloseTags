Auto Close Tags
===============

Sublime Text plugin with sole purpose of extending the built-in auto tag closing functionality.

Formats supported
-----------------

- HTML
- XML
- JSX (Babel, React)

How to use
----------

Tags are automatically closed as you type `</`.

Note that the built-in setting `auto_close_tags` must be enabled.
